# Use Bibliogon on mobile

Bibliogon's editor is published as a web app you can open in any modern
mobile browser and install to your home screen as a Progressive Web App
(PWA) - no app store required.

## Open the web app

Open the following address in your phone or tablet browser:

- [astrapi69.github.io/bibliogon](https://astrapi69.github.io/bibliogon/)

This is the same Bibliogon interface as the desktop app. The
documentation you are reading lives under
[astrapi69.github.io/bibliogon/docs](https://astrapi69.github.io/bibliogon/docs/).

## Install to your home screen

Installing adds a Bibliogon icon to your home screen and launches it
full-screen, without the browser address bar.

- **iOS (Safari):** tap the **Share** button, then **Add to Home
  Screen**.
- **Android (Chrome):** tap the **&#8942;** menu, then **Add to Home
  screen** (or accept the **Install app** prompt when it appears).

## What works today

The web app shares the same code as the desktop app, including the
responsive layout with a hamburger menu on small screens. Opening and
installing the app works in the browser.

## Planned for a future version

> **Planned for a future version.** The items below are not available
> yet. They are listed here so you know what the mobile experience is
> growing towards.

- **Offline authoring.** Writing on the go without a connection -
  drafts stored locally on the device and kept when you close the tab.
- **Sync with your desktop over your local network.** Pairing the
  phone with your desktop Bibliogon (running on the same Wi-Fi) so
  books edited on either device stay in step.

Until then, your books live in your desktop or Docker installation; see
the [Installation overview](../installation.md) for the desktop setup.
