# Formatting

The editor supports bold (Ctrl+B), italic (Ctrl+I), strikethrough (Ctrl+Shift+X), inline code (Ctrl+E), headings H1-H6, bullet and numbered lists, blockquotes, code blocks, images with captions, tables, horizontal rules, footnotes, and links. All formatting can be applied via toolbar buttons or keyboard shortcuts.

Lists can be nested to any depth using Tab for indentation. Images are stored as book assets and support drag-and-drop insertion. Tables can be edited directly in the editor with row and column controls.
