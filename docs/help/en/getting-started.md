# Getting Started

Bibliogon is an open-source book authoring platform. Install it by cloning the repository and running `make install`, or use the Docker setup with `make prod` (port 7880). Start the development server with `make dev` to launch the backend (port 8000) and frontend (port 5173).

Create your first book on the Dashboard by clicking "New Book". Enter a title and author, then add chapters in the editor sidebar. The editor saves your work automatically. You can also import existing projects in write-book-template format (ZIP) or restore backups (.bgb files) from the Dashboard.
