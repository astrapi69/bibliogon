# Bibliogon auf dem Handy nutzen

Der Bibliogon-Editor wird als Web-App veröffentlicht, die du in jedem
modernen mobilen Browser öffnen und als Progressive Web App (PWA) auf
dem Startbildschirm installieren kannst - ganz ohne App Store.

## Web-App öffnen

Öffne die folgende Adresse im Browser deines Smartphones oder Tablets:

- [astrapi69.github.io/bibliogon](https://astrapi69.github.io/bibliogon/)

Das ist dieselbe Bibliogon-Oberfläche wie in der Desktop-App. Die
Dokumentation, die du gerade liest, liegt unter
[astrapi69.github.io/bibliogondocs](https://astrapi69.github.io/bibliogondocs/).

## Auf dem Startbildschirm installieren

Beim Installieren wird ein Bibliogon-Symbol zum Startbildschirm
hinzugefügt und die App im Vollbild ohne Browser-Adressleiste gestartet.

- **iOS (Safari):** Auf **Teilen** tippen, dann **Zum Home-Bildschirm**.
- **Android (Chrome):** Auf das Menü **&#8942;** tippen, dann **Zum
  Startbildschirm hinzufügen** (oder die **App installieren**-
  Aufforderung bestätigen, sobald sie erscheint).

## Was heute funktioniert

Die Web-App teilt sich den Code mit der Desktop-App, einschließlich des
responsiven Layouts mit Hamburger-Menü auf kleinen Bildschirmen. Das
Öffnen und Installieren der App funktioniert im Browser.

## Geplant für eine zukünftige Version

> **Geplant für eine zukünftige Version.** Die folgenden Punkte sind noch
> nicht verfügbar. Sie stehen hier, damit du weißt, wohin sich die
> mobile Nutzung entwickelt.

- **Offline schreiben.** Unterwegs ohne Verbindung schreiben - Entwürfe
  werden lokal auf dem Gerät gespeichert und bleiben erhalten, wenn du
  den Tab schließt.
- **Mit dem Desktop im lokalen Netzwerk synchronisieren.** Das Handy mit
  deinem Desktop-Bibliogon koppeln (im selben WLAN), sodass auf beiden
  Geräten bearbeitete Bücher im Gleichstand bleiben.

Bis dahin liegen deine Bücher in deiner Desktop- oder Docker-
Installation; die [Installations-Übersicht](../installation.md) erklärt
die Desktop-Einrichtung.
