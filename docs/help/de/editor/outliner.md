# Gliederung

Die **Gliederung** ist eine Tabellenansicht der Kapitel eines Buches —
eine Zeile pro Kapitel, mit sortierbaren, direkt bearbeitbaren Spalten.
Sie ist das tabellarische Gegenstück zum kartenbasierten
[Storyboard](../books/storyboard.md): das Storyboard dient dem visuellen
Pacing, die Gliederung dem Überblick und dem schnellen Anpassen von
Metadaten über das ganze Buch hinweg.

## Gliederung öffnen

Klicke in einem Prosa-Buch auf den **Gliederung**-Button in der
Kapitel-Sidebar. Die URL wechselt zu `?view=outline` — du kannst also
direkt verlinken oder mit dem Browser-Zurück-Button zurück in den
Editor.

## Spalten

| Spalte | Bearbeitbar | Hinweise |
|---|---|---|
| **#** | — | Position des Kapitels. |
| **Titel** | — | Anklicken öffnet das Kapitel im Editor. |
| **Wörter** | — | Live-Wortzahl; wird grün, wenn das Kapitel sein Ziel erreicht. |
| **Ziel** | ja | [Wortziel](writing-goals.md) pro Kapitel. |
| **Status** | ja | Bearbeitungs-Status (Zu erledigen / Erster Entwurf / Überarbeitet / Fertig). |
| **Label** | ja | Eines der [Labels](../books/storyboard.md) des Buches. |
| **Beat** | ja | Story-Beat (Exposition … Auflösung). |

Alle Änderungen speichern sofort (dieselbe optimistisch gesperrte
Speicherung wie Storyboard und Editor).

## Sortieren

Klicke auf eine Spaltenüberschrift, um danach zu sortieren; ein
weiterer Klick kehrt die Richtung um. Sortieren ist möglich nach
Position (Standard), Titel, Wortzahl, Ziel oder Status — praktisch, um
etwa die kürzesten Kapitel zu finden oder alles zusammenzufassen, was
noch auf *Zu erledigen* steht. Die Sortierung ist nur eine Ansicht;
sie ändert nie die tatsächliche Reihenfolge der Kapitel.
