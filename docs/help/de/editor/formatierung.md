# Formatierung

## Textformatierung

Bibliogon unterstuetzt die gaengigen Textformatierungen, die du aus Textverarbeitungsprogrammen kennst. Alle Formatierungen sind ueber die Toolbar oder Tastenkuerzel erreichbar:

- **Fett** (Ctrl+B): Hebt wichtige Begriffe oder Saetze hervor.
- **Kursiv** (Ctrl+I): Wird fuer Betonungen, Titel oder fremdsprachige Begriffe verwendet.
- **Durchgestrichen** (Ctrl+Shift+X): Markiert geloeschten oder ungueltig gewordenen Text.
- **Inline-Code** (Ctrl+E): Formatiert Codeabschnitte innerhalb eines Absatzes in Monospace-Schrift.
- **Hochgestellt / Tiefgestellt**: Fuer Fussnotenverweise, chemische Formeln oder mathematische Ausdruecke.

## Ueberschriften

Der Editor unterstuetzt sechs Ueberschriftenebenen (H1 bis H6). In der Praxis genuegen fuer die meisten Buecher drei Ebenen:

- **H1** (Ctrl+Shift+1): Kapitelueberschrift. Wird in der Regel nur einmal pro Kapitel verwendet.
- **H2** (Ctrl+Shift+2): Unterabschnitt innerhalb eines Kapitels.
- **H3** (Ctrl+Shift+3): Unter-Unterabschnitt.

Die Ueberschriften werden beim Export automatisch in das Inhaltsverzeichnis uebernommen. Die Tiefe des Inhaltsverzeichnisses (wie viele Ebenen angezeigt werden) laesst sich in den Export-Einstellungen konfigurieren.

## Listen

Bibliogon unterstuetzt zwei Listenarten:

- **Aufzaehlungsliste** (Ctrl+Shift+8): Ungeordnete Liste mit Aufzaehlungspunkten. Einrueckungen ueber Tab erzeugen verschachtelte Listen.
- **Nummerierte Liste** (Ctrl+Shift+9): Geordnete Liste mit automatischer Nummerierung. Die Nummerierung passt sich automatisch an, wenn du Eintraege hinzufuegst oder entfernst.

Listen koennen beliebig tief verschachtelt werden. Jede Einrueckungsebene erzeugt eine Unterliste.

## Zitate und Code

- **Blockzitat** (Ctrl+Shift+B): Laengere Zitate werden eingerueckt und visuell abgesetzt dargestellt. Blockzitate eignen sich fuer woertliche Rede, Zitate aus anderen Werken oder hervorgehobene Textpassagen.
- **Code-Block**: Fuer laengere Code-Beispiele steht ein Code-Block zur Verfuegung. Der Code wird in Monospace-Schrift angezeigt und Leerzeichen werden exakt beibehalten. Code-Bloecke unterstuetzen Syntax-Highlighting fuer verschiedene Programmiersprachen.

## Bilder und Tabellen

Bilder lassen sich ueber die Toolbar oder per Drag-and-Drop in den Editor einfuegen. Jedes Bild wird als Asset im Buch gespeichert und kann mit einer Bildunterschrift (Figcaption) versehen werden. Unterstuetzte Formate sind PNG, JPG und SVG.

Tabellen koennen ueber die Toolbar eingefuegt und direkt im Editor bearbeitet werden. Du kannst Zeilen und Spalten hinzufuegen oder entfernen, und jede Zelle individuell formatieren.

## Weitere Elemente

- **Trennlinie**: Eine horizontale Trennlinie zwischen Abschnitten.
- **Fussnoten**: Fussnotenverweise im Text mit zugehoerigem Fussnotentext am Seitenende.
- **Links**: Text kann mit URLs verlinkt werden. Der Link-Dialog erlaubt die Eingabe von URL und optionalem Titel.
