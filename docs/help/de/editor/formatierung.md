# Formatierung

## Textformatierung

Bibliogon unterstuetzt die gaengigen Textformatierungen, die du aus Textverarbeitungsprogrammen kennst. Alle Formatierungen sind über die Toolbar oder Tastenkürzel erreichbar:

- **Fett** (Ctrl+B): Hebt wichtige Begriffe oder Saetze hervor.
- **Kursiv** (Ctrl+I): Wird für Betonungen, Titel oder fremdsprachige Begriffe verwendet.
- **Durchgestrichen** (Ctrl+Shift+X): Markiert gelöschten oder ungueltig gewordenen Text.
- **Inline-Code** (Ctrl+E): Formatiert Codeabschnitte innerhalb eines Absatzes in Monospace-Schrift.
- **Hochgestellt / Tiefgestellt**: Für Fußnotenverweise, chemische Formeln oder mathematische Ausdruecke.

## Überschriften

Der Editor unterstuetzt sechs Überschriftenebenen (H1 bis H6). In der Praxis genuegen für die meisten Buecher drei Ebenen:

- **H1** (Ctrl+Shift+1): Kapitelüberschrift. Wird in der Regel nur einmal pro Kapitel verwendet.
- **H2** (Ctrl+Shift+2): Unterabschnitt innerhalb eines Kapitels.
- **H3** (Ctrl+Shift+3): Unter-Unterabschnitt.

Die Überschriften werden beim Export automatisch in das Inhaltsverzeichnis übernommen. Die Tiefe des Inhaltsverzeichnisses (wie viele Ebenen angezeigt werden) laesst sich in den Export-Einstellungen konfigurieren.

## Listen

Bibliogon unterstuetzt zwei Listenarten:

- **Aufzählungsliste** (Ctrl+Shift+8): Ungeordnete Liste mit Aufzählungspunkten. Einrueckungen über Tab erzeugen verschachtelte Listen.
- **Nummerierte Liste** (Ctrl+Shift+9): Geordnete Liste mit automatischer Nummerierung. Die Nummerierung passt sich automatisch an, wenn du Eintraege hinzufuegst oder entfernst.

Listen koennen beliebig tief verschachtelt werden. Jede Einrueckungsebene erzeugt eine Unterliste.

## Zitate und Code

- **Blockzitat** (Ctrl+Shift+B): Längere Zitate werden eingerueckt und visuell abgesetzt dargestellt. Blockzitate eignen sich für woertliche Rede, Zitate aus anderen Werken oder hervorgehobene Textpassagen.
- **Code-Block**: Für laengere Code-Beispiele steht ein Code-Block zur Verfuegung. Der Code wird in Monospace-Schrift angezeigt und Leerzeichen werden exakt beibehalten. Code-Bloecke unterstuetzen Syntax-Highlighting für verschiedene Programmiersprachen.

## Bilder und Tabellen

Bilder lassen sich über die Toolbar oder per Drag-and-Drop in den Editor einfügen. Jedes Bild wird als Asset im Buch gespeichert und kann mit einer Bildunterschrift (Figcaption) versehen werden. Unterstuetzte Formate sind PNG, JPG und SVG.

Tabellen koennen über die Toolbar eingefuegt und direkt im Editor bearbeitet werden. Du kannst Zeilen und Spalten hinzufügen oder entfernen, und jede Zelle individuell formatieren.

## Weitere Elemente

- **Trennlinie**: Eine horizontale Trennlinie zwischen Abschnitten.
- **Fußnoten**: Fußnotenverweise im Text mit zugehoerigem Fußnotentext am Seitenende.
- **Links**: Text kann mit URLs verlinkt werden. Der Link-Dialog erlaubt die Eingabe von URL und optionalem Titel.
