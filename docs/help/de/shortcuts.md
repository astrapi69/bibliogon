# Tastenkürzel

## Editor-Tastenkürzel

Die folgende Tabelle listet alle Tastenkürzel auf, die im TipTap-Editor verfügbar sind. Auf macOS wird Ctrl durch Cmd ersetzt.

| Tastenkürzel | Aktion |
|---|---|
| Ctrl+B | Fett |
| Ctrl+I | Kursiv |
| Ctrl+Shift+X | Durchgestrichen |
| Ctrl+E | Inline-Code |
| Ctrl+Shift+1 | Überschrift 1 |
| Ctrl+Shift+2 | Überschrift 2 |
| Ctrl+Shift+3 | Überschrift 3 |
| Ctrl+Shift+8 | Aufzählungsliste |
| Ctrl+Shift+9 | Nummerierte Liste |
| Ctrl+Shift+B | Blockzitat |
| Ctrl+Z | Rückgängig |
| Ctrl+Shift+Z | Wiederholen |

## Hinweise

Die Tastenkürzel funktionieren nur, wenn der Editor-Bereich den Fokus hat. In Dialogen oder der Sidebar gelten die Standard-Browser-Tastenkürzel.

Für Überschriften 4 bis 6 stehen keine Standard-Tastenkürzel zur Verfuegung. Diese Überschriftenebenen koennen über die Toolbar oder im Markdown-Modus gesetzt werden.

Alle Formatierungen lassen sich auch über die Toolbar anwenden. Der Tooltip jeder Toolbar-Schaltflaeche zeigt das zugehoerige Tastenkürzel an, sofern vorhanden.
