# Tastenkuerzel

## Editor-Tastenkuerzel

Die folgende Tabelle listet alle Tastenkuerzel auf, die im TipTap-Editor verfuegbar sind. Auf macOS wird Ctrl durch Cmd ersetzt.

| Tastenkuerzel | Aktion |
|---|---|
| Ctrl+B | Fett |
| Ctrl+I | Kursiv |
| Ctrl+Shift+X | Durchgestrichen |
| Ctrl+E | Inline-Code |
| Ctrl+Shift+1 | Ueberschrift 1 |
| Ctrl+Shift+2 | Ueberschrift 2 |
| Ctrl+Shift+3 | Ueberschrift 3 |
| Ctrl+Shift+8 | Aufzaehlungsliste |
| Ctrl+Shift+9 | Nummerierte Liste |
| Ctrl+Shift+B | Blockzitat |
| Ctrl+Z | Rueckgaengig |
| Ctrl+Shift+Z | Wiederholen |

## Hinweise

Die Tastenkuerzel funktionieren nur, wenn der Editor-Bereich den Fokus hat. In Dialogen oder der Sidebar gelten die Standard-Browser-Tastenkuerzel.

Fuer Ueberschriften 4 bis 6 stehen keine Standard-Tastenkuerzel zur Verfuegung. Diese Ueberschriftenebenen koennen ueber die Toolbar oder im Markdown-Modus gesetzt werden.

Alle Formatierungen lassen sich auch ueber die Toolbar anwenden. Der Tooltip jeder Toolbar-Schaltflaeche zeigt das zugehoerige Tastenkuerzel an, sofern vorhanden.
