# Tastenkuerzel

Die folgenden Tastenkuerzel sind in Bibliogon verfuegbar. Auf macOS wird Ctrl durch Cmd ersetzt.

## App

| Tastenkuerzel | Aktion |
|---|---|
| Ctrl+/ | Tastenkuerzel anzeigen |
| Ctrl+H | Suchen und Ersetzen (im Editor) |

## Editor

Diese Tastenkuerzel funktionieren, wenn der Editor den Fokus hat.

| Tastenkuerzel | Aktion |
|---|---|
| Ctrl+B | Fett |
| Ctrl+I | Kursiv |
| Ctrl+U | Unterstrichen |
| Ctrl+Shift+X | Durchgestrichen |
| Ctrl+E | Inline-Code |
| Ctrl+Shift+1 | Ueberschrift 1 |
| Ctrl+Shift+2 | Ueberschrift 2 |
| Ctrl+Shift+3 | Ueberschrift 3 |
| Ctrl+Shift+8 | Aufzaehlungsliste |
| Ctrl+Shift+9 | Nummerierte Liste |
| Ctrl+Shift+B | Blockzitat |
| Ctrl+Z | Rueckgaengig |
| Ctrl+Y | Wiederholen |

## Hinweise

Fuer Ueberschriften 4 bis 6 stehen keine Standard-Tastenkuerzel zur Verfuegung. Diese Ueberschriftenebenen koennen ueber die Toolbar oder im Markdown-Modus gesetzt werden.

Alle Formatierungen lassen sich auch ueber die Toolbar anwenden. Der Tooltip jeder Toolbar-Schaltflaeche zeigt das zugehoerige Tastenkuerzel an.
