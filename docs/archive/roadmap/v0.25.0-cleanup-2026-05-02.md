# ROADMAP Cleanup (Archived)

Archived: 2026-05-02
Covers: items completed during Phase 2 between v0.14.0 and v0.25.0
inclusive, plus a few earlier-Phase-2 bugfixes that the Phase 1
archive predated.

This document preserves every closed task that was still listed in
the active ``docs/ROADMAP.md`` and ``docs/backlog.md`` immediately
before the 2026-05-02 cleanup pass. For the current work plan, see
[docs/ROADMAP.md](../ROADMAP.md).

## Summary

Phase 2 turned a feature-complete single-user tool into a
cross-platform distributable, deepened the import pipeline (core
orchestrator with 8 handlers + git adoption), wired a full
git-based backup feature plus a separate plugin-git-sync flow,
shipped article authoring (Article entity + Publications + drift
detection + multi-format export), introduced the donations
surface, finished a 22-component CSS-Modules migration, and ran
two real data-loss-prevention sweeps (DB tripwire after the April
2026 incident; filesystem tripwire on 2026-05-02). The session
also cleared 8 of 9 deferred dependency upgrades (React 18→19,
react-router 6→7, Vite 6→7 + TS 5→6, lucide 0.x→1.x, pandas 2→3,
Pillow 11→12) and migrated to Node 22 LTS.
