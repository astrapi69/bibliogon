# Bibliogon v0.25.0 — 2026-05-01

Articles reach feature parity with books across the full lifecycle: dashboard chrome, soft-delete + trash, AI-generated SEO metadata, backup format extended to manifest 2.0 with article + publication + asset segments, and CIO-handler restore through the Import Wizard. Three-layer secrets configuration (project YAML < user override < env-var) replaces the old "edit `app.yaml`" advice with a Gradle-style override file. The donations theme ships its first user-visible surface (S-01 Settings tab "Unterstützen" with four-channel grid). T-01 inline-styles refactor migrates 22 components / pages to per-file CSS Modules, eliminating ~700 inline-style call-sites. Plus a mobile-first hamburger for the Settings tabs (F-01) and a sturdier `make dev-bg` that no longer dies silently when the recipe shell exits (F-02).

## Action required

No new database migrations.

If you carry a real Anthropic API key in `backend/config/app.yaml`, move it to `~/.config/bibliogon/secrets.yaml` (or set `BIBLIOGON_AI_API_KEY`) per [docs/configuration.md](https://github.com/astrapi69/bibliogon/blob/main/docs/configuration.md); the running app still reads the old location with a deprecation warning, but the Settings UI hides the API-key input so editing it requires the override path.

The donations block was already shipped in `app.yaml.example` in v0.24.0 cycle but was inert; v0.25.0 wires it into the UI. Existing installs gain the "Unterstützen" tab automatically once their `app.yaml` carries the block (copy from `app.yaml.example` if missing).

## Highlights

- **Articles dashboard parity.** Soft-delete + trash, AI-generated SEO, list/grid view toggle, action menu, header chrome, refresh on bfcache.
- **Articles in backup format (manifest 2.0).** Export + CIO restore + Import Wizard treat articles as first-class. Idempotent re-import; soft-delete revival.
- **Three-layer secrets configuration.** Project YAML < user override < env-var. AiSetupWizard + Settings UI hide the API-key input when an override is configured.
- **Donations S-01 — Settings "Unterstützen" tab.** Four-channel grid (Liberapay, GitHub Sponsors, Ko-fi, PayPal) with `recommended` badge.
- **T-01 inline-styles refactor.** 22 components / pages migrated to CSS Modules; ~700 inline-style call-sites eliminated.
- **Settings mobile hamburger (F-01).** Tabs.List collapses into a `Menu`-icon dropdown below 768px; same `tabDefs` array drives both surfaces.
- **`make dev-bg` robustness (F-02).** `setsid` + log file capture + `kill -0` startup probe. New `make dev-bg-logs` tails both files.
- **AI client now uses the new loader.** Fixes the regression where `ai/routes.py` bypassed the override file.
- **Centralised app version.** Single source of truth in `backend/app/__init__.py`; replaces a stale `0.15.0` hardcode in the health endpoint.
