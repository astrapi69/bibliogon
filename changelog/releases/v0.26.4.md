# Bibliogon v0.26.4 — 2026-05-04

Hotfix for v0.26.3. v0.26.3 attached only the macOS launcher binary; Linux and Windows release-event runs failed. v0.26.4 fixes both and is the version users on Linux or Windows should install.

## What broke in v0.26.3

The inline release-event verify step in the launcher workflows was positioned before `actions/setup-python`. On Linux this meant `python3` resolved to ubuntu-22.04's system Python 3.10, which lacks `tomllib` (stdlib in 3.11+). On Windows, `sync_versions.py` invoked `generate_install_sh.sh` via `subprocess.run([str(script), '--check'])`, which raises `WinError 193` because Windows cannot natively exec `.sh` files via subprocess.

## Fixes in v0.26.4

- All three launcher workflows now run the verify step AFTER `actions/setup-python` + `Install Poetry`, guaranteeing Python 3.12 with `tomllib`.
- `sync_versions.py:regenerate_install_sh` invokes `bash` explicitly (`subprocess.run(['bash', str(script), '--check'])`) so the same code path works on Linux, macOS, and Windows runners (Git Bash is pre-installed on the GitHub Actions Windows runner).

No production behavior change. v0.26.4 ships the same substantive work as v0.26.0 (foundation cleanup; see [v0.26.3 release notes](https://github.com/astrapi69/bibliogon/releases/tag/v0.26.3) for the full picture).

## Hotfix history

- `v0.26.0`: original tag; release-gate failed (chmod bit).
- `v0.26.1`: chmod fix; launcher builds failed (PyInstaller spec `__file__`); CI failed (mypy).
- `v0.26.2`: spec + mypy fixes; CI failed (ruff format).
- `v0.26.3`: ruff format applied; macOS launcher attached; Linux + Windows release-event runs failed (Python step order, Windows subprocess).
- `v0.26.4`: workflow + subprocess fixes. All three launcher binaries attached.

The cluster prompted a release-workflow rule update (commit 627a6b7) making the local pre-tag verify chain MANDATORY: ruff, mypy, pre-commit, AND `pyinstaller bibliogon-launcher.spec --clean --noconfirm`. Future hotfix clusters of this size are blocked at the local sweep before they can reach CI.
